Duct

A server-side framework for Clojure

Define the structure of your application as pure data in duct.edn. 

{:system
 {:duct.module/logging {}
  :duct.module/sql {:migrations #duct/include "db/migrations.edn"}
  :duct.module/web {:features #{:api}
                    :routes [["/items" #ig/ref :demo.routes/items]]}
  :demo.routes/items {:db #ig/ref :duct.database/sql}}}

Modules remove repetition while still allowing you to override any data they add.

Use functions and multimethods to define the behavior of your components. 

(ns demo.routes
  (:require [next.jdbc :as jdbc]))

(defn list-items [{:keys [db]}]
  (fn handler [_request]
    {:body {:results (jdbc/execute! db ["SELECT * FROM items"])}}))
